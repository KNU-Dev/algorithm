# algorithm

KNU-Dev 알고리즘 문제 풀이 저장소입니다.

## 구조

각자 자신의 이름 디렉터리 아래에 `플랫폼/난이도/문제명/문제.java` 형식으로 풀이를 올려주세요.

```
algorithm/
└── 최승원/
    └── 프로그래머스/
        └── 0/
            └── 문제명/
                └── 문제.java
```

## 규칙

- 각자 본인 디렉터리에만 파일을 추가/수정합니다.
- 난이도 폴더는 `0`, `1`, `2` 중 하나를 사용합니다.

## 브랜치 및 PR

- `main`에 직접 push하지 않고, 본인 이름으로 브랜치를 만들어 작업합니다. (예: `seungwon-choi`)
- 작업이 끝나면 해당 브랜치에서 `main`으로 PR을 올립니다.
